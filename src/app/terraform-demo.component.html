Terraform Resource with Sensitive Properties Demo This demonstrates masking sensitive property values from Terraform plan JSON

How it works:

  • Properties marked as sensitive in after_sensitive are masked with ****
  • Click the eye icon to toggle visibility of sensitive values
  • Array elements can have individual sensitivity flags
  • In this example:
    • custom_domain_verification_id is sensitive
    • triggers_replace[0-2] are sensitive
    • triggers_replace[3-4] are NOT sensitive
